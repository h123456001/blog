swift 3.0 Block

闭包

{ (parameters) -> (return type) in
statements
}

是可以在你的代码中被传递和引用的功能性独立模块。Swift 中的闭包和 C 以及 Objective-C 中的 blocks 很像，还有其他语言中的匿名函数也类似。
e.g:
sorted(by:)

let names = [“Chris”,”Alex”,”Ewa”,”Barry”,”Daniella”]
此处排序需要传入(string,string)->bool
普通方法：

names.sorted(by:{(str1:String,str2:String)->Bool in
return str1<str2
})
```
